## 📊 RECOMENDACIÓN FINAL

### ✅ MEJOR OPCIÓN (Google Fonts Gratis):

```css
/* Headings - Máxima similitud con Nexa */
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@600;700;800;900&display=swap');

/* Body - Legibilidad en pantalla */
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&display=swap');

h1, h2, h3, h4, h5, h6 {
  font-family: 'Poppins', sans-serif;
  font-weight: 700;  /* Similar a Nexa Bold */
  letter-spacing: -0.02em;  /* Nexa usa condensing */
}

p, body {
  font-family: 'Inter', sans-serif;
  font-weight: 400;
  line-height: 1.6;
}
```

## ⚡ IMPLEMENTACIÓN INMEDIATA

```html
<!-- Reemplazar esto: -->
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Outfit:wght@300;400;700;900&display=swap">

<!-- Con esto: -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Poppins:wght@600;700;800;900&family=Inter:wght@400;500;600&display=swap">
```

```css
/* En tu <style> tag: */
h1, h2, h3, h4, h5, h6, .font-brand-bold {
  font-family: 'Poppins', sans-serif;
  font-weight: 700;
  letter-spacing: -0.02em;
}

body, p {
  font-family: 'Inter', sans-serif;
}
```

## ❓ PREGUNTAS FRECUENTES

**P: ¿Outfit está mal?**
> R: No está "mal", pero es demasiado suave (humanista). Nexa es geométrica. Visualización totalmente diferente.

**P: ¿Verá el cliente la diferencia?**
> R: Sí. Poppins se ve más "premium" y "moderno" como Nexa. Outfit se ve más "friendly" y "casual".

**P: ¿Y si compro Nexa en Adobe Fonts?**
> R: Buena idea si tienes presupuesto. Entonces úsala 100% (headings + body) con Inter como fallback.

**P: ¿Puedo mezclar 3 fuentes?**
> R: Técnicamente sí, pero es excesivo. Poppins + Inter = equilibrio perfecto.

---

**Conclusión:** Usa **Poppins + Inter**. Te ahorra plata y se parece 85% a Nexa. Imperceptible al usuario final.
